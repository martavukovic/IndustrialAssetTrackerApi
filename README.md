# 🏭 Industrial Asset Tracker - Full Stack

A comprehensive management system for tracking industrial equipment, assets, and maintenance schedules. This project demonstrates a modern Full Stack architecture with a decoupled Backend and Frontend.

---

## 🚀 Tech Stack

### Backend
- **Framework:** .NET 8 Web API
- **Database:** Entity Framework Core (SQL Server)
- **Architecture:** RESTful API with Controller-based routing

### Frontend
- **Framework:** React 18 (Vite)
- **Styling:** Tailwind CSS
- **HTTP Client:** Axios for API communication

---

## 📁 Project Structure

The repository is organized into two main modules:

- **`IndustrialAssetTrackerApi/`** - Contains the C# backend logic, database models, and API endpoints.
- **`IndustrialAssetTrackerFrontend/`** - Contains the React application, UI components, and state management.
